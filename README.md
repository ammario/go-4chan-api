A 4chan API client for Go. Supports:
- API revision 23 (4 Jan 2013)
	* Single thread
	* Thread index
	* Board list
	* Board catalog
- HTTPS
- Rate limiting
- `If-Modified-Since`

[Examples and docs on GoDoc.](http://godoc.org/github.com/moshee/go-4chan-api)

Pull requests welcome.
